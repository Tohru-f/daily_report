## わかったこと
- Node.jsでツールをインストールする方法は２種類あり、グローバルインストールとローカルインストールがある。同じツールの別バージョンを複数の開発プロジェクトで使えるよう基本的にツールはローカルインストールするべきである。
- Nodejsにおいて、パッケージ(ライブラリやツール)には本番環境でも使いたいものと開発中だけ使いたいものに分類され、package.json上で前者はdependenciesへ、後者はdevDevdependenciesへ記入される。
- 本番環境でも使う場合はnpm install パッケージ名、テスト環境だけで使う場合はnpm install --save-dev パッケージ名でインストールする。
- ローカルインストールしたNodejsのツールはnpxコマンドで起動する。
- package.jsonのscripts部分はツールを起動するための短縮コマンドにあたる。npm run 設定したコマンド でツールを起動できる。
